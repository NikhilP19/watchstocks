# watchstocks

Track stocks

#### [Demo](http://watchstocks.herokuapp.com/)

## Run

Install dependencies

```
npm i && bower i
```

MongoDB is required on localhost, otherwise change config in
```
server/config/environment/development.js
```

then run
```
grunt serve
```

## Uses

- NodeJS
- Socket.io
- AngularJS
- Highcharts
- SCSS
- Karma
- Protractor
- Yeoman
